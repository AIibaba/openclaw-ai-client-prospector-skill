*Professional B2B lead generation system — discover high-value prospects, qualify leads, craft personalized outreach, and build your sales pipeline.*

An intelligent AI-powered client acquisition engine designed for freelancers, agencies, and B2B businesses. It automates the entire prospecting workflow from identifying ideal customer profiles to generating personalized outreach messages that convert.

## Features

| Feature | Description |
|---------|-------------|
| Smart Lead Discovery | Identifies high-value prospects based on industry, company size, tech stack, and buying signals |
| Lead Qualification Engine | Scores and prioritizes leads using multi-dimensional criteria to focus on the highest-converting opportunities |
| Personalized Outreach | Generates tailored cold emails, LinkedIn messages, and follow-up sequences for each prospect |
| ICP Builder | Helps define and refine your Ideal Customer Profile based on your service offerings and past wins |
| Pipeline Management | Tracks prospect status, follow-up schedules, and conversion metrics |
| Multi-Channel Strategy | Supports outreach across email, LinkedIn, Twitter, and community platforms |

## Getting Started

```bash
# Install via OpenClaw
openclaw install AIibaba/openclaw-ai-client-prospector-skill
```

## Usage

```
build icp               # Define your Ideal Customer Profile
find leads [industry]    # Discover prospects matching your ICP
qualify [company]        # Score and assess a specific lead
write outreach [lead]    # Generate personalized outreach messages
pipeline report          # View your sales pipeline status
```
